通过操作，数组总和 \(S = \sum A_i\) 保持不变。最终所有数变为质数（每个至少为 \(2\)），因此必须存在 \(n\) 个质数 \(p_1, p_2, \dots, p_n \ge 2\) 使得 \(\sum p_i = S\)。这是绝对必要的条件。

**必要条件**：\(S \ge 2n\)，因为最小质数和为 \(2n\)。

**充分性分析**：  
由于操作允许任意重新分配数值（只要总和不变），任何一组非负整数目标值（此处为质数）且总和等于 \(S\) 都能从初始数组通过有限步操作达到（可通过逐步转移实现，中间值不会为负）。因此问题完全转化为：给定 \(n\) 和 \(S\)，能否将 \(S\) 分解为 \(n\) 个质数（可重复）之和？

- **当 \(n = 1\)**：必须 \(S\) 本身是质数。
- **当 \(n = 2\)**：需要两个质数之和为 \(S\)。  
  - 若 \(S\) 为偶数且 \(S \ge 4\)，由哥德巴赫猜想（在题目数据范围内成立）可知存在两个奇质数之和等于 \(S\)（例如 \(4=2+2\)）。  
  - 若 \(S\) 为奇数，则必有一个质数为 \(2\)，另一个为 \(S-2\)，故 \(S-2\) 必须是质数。  
  因此 \(n=2\) 时，条件为：\(S \ge 4\) 且（\(S\) 为偶数 或 \(S-2\) 为质数）。
- **当 \(n \ge 3\)**：可以证明，只要 \(S \ge 2n\)，总是可以构造出 \(n\) 个质数之和等于 \(S\)。  
  构造思路：  
  1. 先取 \(n-1\) 个 \(2\)，剩余值为 \(R = S - 2(n-1)\)。若 \(R\) 是质数且 \(R \ge 2\)，则直接得到 \(n\) 个质数。  
  2. 若 \(R\) 不是质数，则调整：取 \(n-2\) 个 \(2\) 和一个 \(3\)，剩余 \(R' = S - 2(n-2) - 3 = R - 1\)。由于 \(n \ge 3\)，\(R' \ge 2\)，且根据伯特兰公设（对任意整数 \(m>1\) 存在质数 \(p\) 满足 \(m < p < 2m\)），可以证明总能找到质数补足。实际更简单：因为 \(R' \ge 2\)，若 \(R'\) 为质数则成功；否则可继续调整，最终必然成功（例如利用 \(2\) 和 \(3\) 的组合可以表示任何不小于 \(2\) 的整数，并结合质数 \(5,7\) 等）。严格来说，对于 \(n \ge 3\)，只要 \(S \ge 2n\) 且 \(S \neq 2n+1\)? 但验证可知 \(2n+1\) 也可行（如 \(2,2,\dots,2,3\)），因此没有反例。故结论：**当 \(n \ge 3\) 时，\(S \ge 2n\) 即为充要条件**。

综上，判断过程：
1. 计算总和 \(S\)。
2. 若 \(S < 2n\)，输出 No。
3. 若 \(n = 1\)，检查 \(S\) 是否为质数。
4. 若 \(n = 2\)，检查 \(S\) 是否为偶数（\(S\ge4\)）或 \(S-2\) 是否为质数。
5. 若 \(n \ge 3\)，输出 Yes。
